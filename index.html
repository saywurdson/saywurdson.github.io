TX Workers' Comp Analytics

↗

Data Pipeline · OMOP CDM · Dashboard

Socrata API → dlt → DuckDB → dbt → OMOP CDM v5.4 → Streamlit

End-to-end pipeline that extracts Texas workers' compensation medical billing data from the Texas Open Data Portal, transforms it through a 3-layer dbt architecture with 48 SQL models, and loads it into the OMOP Common Data Model. Maps ICD-9/10 → SNOMED, NDC → RxNorm, and HCPCS/CPT → standard concepts. Includes a 7-tab Streamlit dashboard covering injury profiles, cost analytics, opioid monitoring, provider networks, and geographic distribution.

Synthetic Data · OMOP CDM · Tuva

Synthetic Medicare patient cohort generated with Synthea, loaded into both the OMOP CDM (via OHDSI/dbt-synthea) and the Tuva health data framework using DuckDB.
